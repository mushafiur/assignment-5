What is the difference between var, let, and const?
ans: var, let, and const are used to declare variables in JavaScript.

var: Old way to declare variables. It can be redeclared and reassigned, and it is function scoped.

let: Modern way. It cannot be redeclared in the same scope, but it can be reassigned. It is block scoped.

const: Used for variables that should not change. It cannot be redeclared or reassigned, and it is also block scoped.


What is the spread operator (...)?

ans: The spread operator (...) in JavaScript is used to expand elements of an array or object into individual values.



What is the difference between map(), filter(), and forEach()?

ans: map(): Creates a new array by applying a function to each element.

filter(): Creates a new array with only the elements that pass a condition.

forEach(): Loops through each element, but does not return a new array.

What is an arrow function?
An arrow function is a shorter way to write a function in JavaScript using the => syntax.


What are template literals?
Template literals are a way to write strings in JavaScript using backticks ( ) so you can easily add variables inside a string.
